# 👨‍💼 Employee Management System

A modern Employee Management System developed to simplify employee record management. The application provides an intuitive interface for managing employee information, departments, attendance, salaries, and other HR-related operations.

---

## 📌 Features

- 👤 Employee Registration
- ✏️ Add, Edit & Delete Employee Records
- 🔍 Search & Filter Employees
- 🏢 Department Management
- 📅 Attendance Tracking
- 💰 Salary Management
- 📊 Dashboard with Statistics
- 🔐 Secure Authentication
- 📱 Responsive User Interface
- 🌙 Dark Mode Support (Optional)

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/kriss2012/Employee-Management-System-Design.git
```

### Navigate into the Project

```bash
cd Employee-Management-System-Design
```

### Install Dependencies

```bash
mvn clean install
```

### Configure Database

Update your database credentials in:

```
application.properties
```

Example:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/employee_db
spring.datasource.username=root
spring.datasource.password=your_password
```

### Run the Project

```bash
mvn spring-boot:run
```
